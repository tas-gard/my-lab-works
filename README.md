# Лабораторные работы по программированию (C, Python)

Этот репозиторий содержит выполненные мной лабораторные работы в рамках дисциплин  
**«Фундаментальная информатика»** и **«Алгоритмы и структуры данных»** (1 курс, МАИ).

Все работы реализованы на **C**, оформлены и собираются с помощью **Makefile**, в папке **report** хранится отчет и приложение для каждой работы.

## Структура

- `LR_9` – Целочисленные вычисления  
- `LR_11` – Лексический анализ  
- `LR_12` – Обработка целых чисел  
- `LR_13` – Работа с множествами  
- `LR_14` – Циклы и обход массивов  
- `LR_15` – Обработка матриц  
- `KP_3` – Вещественный тип, вычисление многочленов, разложение по Тейлору  
- `KP_6` – Обработка последовательной файловой структуры  
- `KP_8` – Линейные списки 
- `LR_1_Python` - Объектно-ориентированная реализация структур данных
